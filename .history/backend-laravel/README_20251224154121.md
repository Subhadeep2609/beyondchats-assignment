# BeyondChats Blog Scraper (Laravel)

## Overview
This Laravel backend scrapes blog articles from BeyondChats and exposes them via a REST API.

## Features
- Artisan command to scrape blogs
- Stores articles in SQLite database
- REST API to fetch articles

## Setup Instructions

### 1. Install dependencies
```bash
composer install
```
### 2. Environment setup
```bash
copy .env.example .env
php artisan key:generate
```
### 3. Database
Uses SQLite. Create file:
```bash
database/database.sqlite
php artisan migrate
```
### 4. Run scraper
```bash
php artisan scrape:beyondchats
```
### 5. Run server
```bash
php artisan serve
```
###   API Endpoint
```bash
GET /api/articles
```

Returns scraped articles in JSON format.
